# Contributing (Bootstrap Phase)

This repo is currently a bootstrap/documentation shell for Zenithbuild governance and contracts.

## Allowed changes (for now)
- Governance docs
- Contract docs
- Minimal scaffolding for repo structure

## Not allowed in this phase
- Workspace wiring
- Moving package repos into this repo
- Publish automation or release scripting
- Source-code changes for Zenith packages

If you need to change packages, do so in their existing repos until the workspace plan is finalized.
