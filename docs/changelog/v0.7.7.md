# zenith-docs v0.7.7

Stable patch release for the Zenith framework train.

## Highlights

- Fixed tagged-release CI/publish instability by aligning Playwright browser installation with the exact workspace version used by smoke tests.
- Preserved all existing framework semantics and release gates; this release is workflow-stability focused.

## 🔧 Release Workflow Stability

- Updated CI/release workflow browser installation to run through `apps/smoke-test` (`bun run --cwd apps/smoke-test playwright install --with-deps chromium`).
- Eliminated Playwright revision drift (`installed browser != expected browser`) that was failing smoke checks in tagged publish runs.

## 📦 Published Versions

- Lockstep train `0.7.7` covers `@zenithbuild/bundler`, `@zenithbuild/cli`, `@zenithbuild/compiler`, `@zenithbuild/core`, `@zenithbuild/router`, `@zenithbuild/runtime`, `@zenithbuild/language`, and `@zenithbuild/language-server`.
- Optional platform packages for `@zenithbuild/bundler-*` and `@zenithbuild/compiler-*` move with the same `0.7.7` train.
