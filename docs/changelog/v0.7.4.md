# zenith-docs v0.7.4

## Highlights

- Lockstep release `0.7.4` publishes compiler platform packages whose `zenith-compiler` binary includes `--merge-image-materialization`, so npm installs resolve the same Track B image artifact path as local and CI builds.
- CI continues to gate the staged platform binary (`scripts/verify-compiler-shipped-surface.mjs`) before the rest of the suite.

## Verification

- After publishing, confirm registry versions with:

  `npm view @zenithbuild/compiler@0.7.4 version`

  and optionally `node scripts/verify-npm-registry-train.mjs`.

## Deferred (not this release)

- Broader Track B scope (bundler-owned final HTML materialization, CDN integrations, dynamic image props) remains intentionally out of this train cut.
