# Archived Legacy Compiler Snapshot

This directory is an archived internal snapshot from the legacy Zenith v1 compiler work.

It is not part of the current public Zenith API surface, is not a supported package entrypoint, and must not be used as evidence that Zenith currently exposes a public plugin API. Current Zenith configuration rejects `plugins`; public plugin design belongs to future roadmap work, not this archive.

Do not copy plugin, config, or runtime patterns from this directory into current framework code without a separate approved design issue.
