# Archived Legacy Bundler Snapshot

This directory is an archived internal snapshot from the legacy Zenith v1 bundler work.

It is not part of the current public Zenith API surface and must not be used as evidence that Zenith currently exposes public plugin hooks or a public plugin architecture. Current framework extension behavior is limited to the shipped contracts in the active docs and packages.

Do not copy plugin, config, or runtime patterns from this directory into current framework code without a separate approved design issue.
